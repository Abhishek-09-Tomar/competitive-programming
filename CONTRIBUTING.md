## üìù How to Contribute

1. **Fork** the repository.
2. **Clone** your fork locally:

```bash
git clone https://github.com/your-username/My-CPP-Journey.git
```

1. **Create a branch** for your work:

```bash
git checkout -b feature/add-new-solution
```

1. **Add your solution**, project, or improvement.
2. Follow the **folder structure and naming conventions**.
3. **Commit** your changes:

```bash
git add .
git commit -m "Add [problem name] solution"
```

1. **Push** your branch to your fork:

```bash
git push origin feature/add-new-solution
```

1. **Create a Pull Request** to the main repository.

## üìÇ Folder & File Guidelines

- Use **snake_case** for file names (e.g., `two_sum.cpp`)
- Each solution file should include:
  - Problem link (if applicable)
  - Approach/Algorithm
  - Time Complexity
  - Space Complexity
- Example folder placement:

```
dsa/arrays/two_sum.cpp
competitive_programming/codeforces/contest_123/problem_A.cpp
projects/mini/project_name.cpp
```
